Resume Website

Construction began September 10 2022


This resume was the first project I began working on in September of 2022 as I began classes for Full Stack Development. This resume displays my abilities with HTML5 and CSS at the moment and is constantly being updated to reflect my skills, projects, and work experience.

For this project I used Visual Studio Code and started by creating a fairly barebones single-page website that was not hosted online anywhere. Once I learned about how to use Github Pages to host small projects I put the resume website online and continued adjusting and troubleshooting the CSS styling.

Once I had the styling nearly complete I began to expand the webpage and started adding more pages to turn it into a more comprehensive website so that I could practice building an organized file system. I poked around to try to learn how most people handled CSS styling for multi-page websites and found that smaller projects like this one usually had one comprehensive CSS stylesheet for the entire site.

This project is currently in another cleanup stage where I'll be adjusting CSS styling to better suit the multiple pages I have added to the structure of the site.

Cheat Sheet Website

Construction began October 2 2022

This site is a series of small tables of commonly used HTML and CSS code and how to use those bits of code. It contains links to Mozilla Developer Network pages for more detailed information and a uses a color palette found using the Coolers tool.

This is a webpage that was a project that was assigned to me from the excellent Codecademy online school. It was the first off-platform assignment they had me do. I use from time to time as a quick-reference tool while working on other projects. It is not as detailed or comprehensive as other tools like MDN or W3S, but this particular guide is easier for me to follow because I made it for me.

Currently on the to-do list for this project is expansion and tidying up of some of the styling. There are two or three tables that are nearly non-existant (though their html structure has been laid out) and every table is missing several important bits of information. I'm sure this site will be something I work on for a long time as i learn more and more, similar to the resume site which I will need to keep up to date as well.
